Summary

  1. When both reference data library size and number of genes are small, adjusted estimator has large variance and bias. The high coverage is due to the large standard error, which leads to \([0,1]\) confidence intervals. For example, let’s take a look at reference data library size \(0.5*G\) and \(G=50\), the confidence interval for \(p_1=0.1\),

all genes

genes expressed at extremely high levels in the reference dataset will dominate the inference results. Since highly expressed genes have large variance, our inferences become very sensitive to these outliers. We therefore remove them to acquire more robust estimations. The total removed genes were the union of the top 1 %.
